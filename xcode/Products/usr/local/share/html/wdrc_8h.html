Function to apply dynamic-range compression to a single sub-band of the output of the analysis filter bank.

The peak detector output in dB SPL is needed as one of the inputs. The gain at 50 and 80 dB SPL is specified for the frequency sub-band, along with the lower and upper kneepoints in dB SPL. The compressor is linear below the lower kneepoint and applies compression limiting above the upper kneepoint

Parameters
inputPointer to the signal array at the frequency sub-band (output from sub-band filtering)
pdBPointer to the peak detector output array of the sub-band in dB SPL (output from peak_to_spl)
gain50Gain in dB at the sub-band frequency for an input at 50 dB SPL
gain80Gain in dB at the sub-band frequency for an input at 80 dB SPL
knee_lowLower kneepoint in dB SPL for the sub-band
knee_highUpper kneepoint in dB SPL for the sub-band
lenLength of the input array
outPointer to an array where the compressed output of the sub-band will be written. i.e. the output of WDRC
