Updating u_prefiltered_accumulated buffer which is circular.

Replace the frame_size number of old values with new values in the circular buffer using the last element of the buffer as the start index.

Parameters
u_prefiltered_accumulatedPointer to the buffer of accumulated values of pre-filtered output of u(n) i.e., u_f(n)
new_framePointer to the pre-filtered output of u(n) for the new frame
u_prefiltered_acc_lenLength of the u_prefiltered_accumulated buffer
new_frame_lenLength of new_frame which is equal to the frame_size
