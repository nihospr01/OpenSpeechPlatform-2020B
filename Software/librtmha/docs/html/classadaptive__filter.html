Detailed Description

Adaptive Filter Class.

This adaptive filter class implements several popular LMS-based algorithms including Modified LMS [Greenberg, 1998], IPNLMS-l_0 [Paleologu et al., 2010] and SLMS [Lee et al., 2017].
